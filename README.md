# HTML/Javascript example Bitcoin SV launcher for [SporeStack](https://sporestack.com)

Uses [MoneyButton](https://moneybutton.com/) for payments. This makes affiliates really easy as you can see in the source, which takes a 1% affiliate cut.

index.html and static/sporestack.js are released into the public domain. (Minus the waving OctoCat bit, which is not mine)

Comment form is [bitsqus](https://bitsqus.github.io).
